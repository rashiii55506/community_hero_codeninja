# Civic Guardian - AI Civic Tech Platform

> **Author**: Rashi Sharma

Welcome to **Civic Guardian**, a next-generation civic-tech platform designed to empower citizens and streamline municipal operations. This application leverages artificial intelligence, computer vision, and gamification to create a proactive, secure, and rewarding urban management ecosystem.

## 🚀 Features

### For Citizens
* **AI Report Submission Engine**: Upload images or videos of civic issues (e.g., potholes, water leaks). The integrated computer vision AI automatically detects hazards, evaluates structural integrity, and categorizes the severity level.
* **Smart Voice & Multilingual Input**: Speak in your local language to report issues. Real-time speech-to-text and AI translation instantly convert local dialects into professional English for official records.
* **Civic Gamification & Rewards**: Earn Impact Points (XP) for reporting and verifying issues. Rank up to become a 'Neighborhood Legend' and unlock civic impact vouchers and badges.
* **Live City Grid Map**: Visualize ongoing reports and neighborhood safety indices in a highly interactive UI.

### For Authorities & City Council
* **Smart AI Triage Inbox**: Automatic routing of issues to the correct department (e.g., Public Works, Sanitation) complete with AI notes and severity badges.
* **Predictive Heatmap & Analytics**: Data-driven insights to predict infrastructure decay and optimize city resources.
* **Micro-Tendering**: Streamlined workflows for allocating minor civic repairs to verified contractors.

## 🛠️ Technology Stack
* **Frontend**: React (v18+), TypeScript
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion (for buttery smooth micro-interactions and transitions)
* **Icons**: Lucide React
* **Build Tool**: Vite

## 📦 Getting Started

### Prerequisites
* Node.js (v18 or higher)
* npm or yarn

### Installation
1. Clone the repository
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy
The application embraces a premium **Dark-Mode-First** aesthetic (`bg-slate-950`). It avoids generic templates by strictly relying on deliberate typography, high-contrast states, and neon/emerald accents to convey a sense of elite, official enterprise software while remaining accessible to everyday citizens.
